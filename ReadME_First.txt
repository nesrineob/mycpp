cher collegue pour ce projet myC++,
je vous propose d installer sur vos machines un fork d apt-get ou aptitude
pour installer vos prochaines lib ou brique.
pour cela je vous propose d installer MacPorts
une premiere install par un packet dmg

https://distfiles.macports.org/MacPorts/MacPorts-2.1.2-10.8-MountainLion.pkg

puis
lorsque vous avez finis de faire l install, vous utiliser 
macport comme si vous utiliser apt-get ou aotitude
ex : sous debian l install de build-essential vous faite
sudo apt-get install build-essential

sous mac osX vous allez faire
sudo port install build-essential

voila, pour plus de doc, vous avez : 
http://guide.macports.org/#introduction
